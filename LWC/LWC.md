# Lightning Web Component
Boilerplate for Lightning Web Component.


## Html template
```
<template>
    //slds styling
    <p>Hello, {variableName}!</p>
</template>
```

## JavaScript
```
import { LightningElement } from "lwc";

export default class className extends LightningElement {
  variableName = value;
  //other logic here
}
//External CSS optional
```
